DS2 Modifications

For the purpose of this research, the original DS2 files have been lighlty modified. The jsp implementation for DS2 was utilitzed, so the infrastructure for using the jsp files was modified to fit the needs of the research. The changes include:

  • ds2webdriver code changed to print to standard output instead of standard error
  • Database generation scripts updated to work with MySQL 5.7 (Type ==> Engine syntax change)
  • ds2webdriver compiled with mono so that it runs on Linux machines.
  • General usability (chmod +x all shell scripts)
  • JSP files modified to work with remote MySQL server
  • Cleanup of the root directory
  • Additional dirver config files added

A zipped archive of the modified ds2 files can be downloaded here.
